Pizza Panic
===========
IMPORTANT NOTICE: You will need Python 3.1 and Pygame for Python 3.1.
Running
=======
Windows: Open IDLE 3.1. Click File>>>Open and select main.py. Press Open. Press f5.
Linux/Ubuntu: Open LXTerminal/Terminal. Type ‘cd’ then the directory in which you saved              
              the files. Then type ‘python3 main.py’.
